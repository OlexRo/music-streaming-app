## Описание

[Nest](https://github.com/nestjs/nest) Репозиторий стартеров на TypeScript для фреймворка.

## Установка

```bash
$ npm install
```

## Запуск приложения

```bash
# режим разработки
$ npm run start

# режим отслеживания изменений
$ npm run start:dev

# режим продакшн
$ npm run start:prod
```

## Описание

Проект на [React](https://reactjs.org/) с использованием [TypeScript](https://www.typescriptlang.org/) для создания мощных и типобезопасных приложений.

## Установка

```bash
$ npm install
```

## Запуск приложения

```bash
# режим разработки
$ npm run dev

# сборка проекта
$ npm run build

# предварительный просмотр сборки
$ npm run preview
```
